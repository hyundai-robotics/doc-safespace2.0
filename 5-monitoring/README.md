# 5. 안전 상태 모니터링

안전 기능 위반 및 안전 제어 모듈(SCM: Safety Control Module) 보드의 상태를 모니터링합니다. 로봇 제한 기능과 조인트 제한 기능, SCM의 상태를 확인할 수 있습니다.

1.  **\[설정]** 버튼 > **\[4: 응용 파라미터 > 21: 협동로봇 설정 > 1: 안전 기능 > 5: 안전 상태 모니터링]** 메뉴를 터치하십시오.


2. &#x20;협동로봇의 안전 상태와 안전 기능별 상태를 확인하십시오.
